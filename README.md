# MapReduce System Project

# *Setup Guide*


*1. Run `sudo docker compose up` in the project folder* <br>
*2. Find the database container id and run `make db id={database_cont_id}`* <br>
*3. Run `sudo docker restart`* <br>
*4. Run `make nginx`* <br>
*6. Done*
